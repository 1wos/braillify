# 설치
<br/>

Braillify 는 Node.js(Wasm for Web) 와 Python 환경을 지원합니다.

표준 저장소에 배포되었기 때문에 모든 패키지 매니저로 다운받을 수 있습니다.

<br/>
## Python
<br/>
```bash
pip install braillify # python
```

<br/>
## Node.js
<br/>
```bash
npm install braillify # nodejs
```

<br/>
## Cargo
<br/>
```bash
cargo add braillify # cargo
```
